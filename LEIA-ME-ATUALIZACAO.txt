ATUALIZAÇÃO DE ESTRUTURA DE CÓDIGO

Esta versão reorganiza o Meu Fluxo em módulos para facilitar manutenção.

Para atualizar sem perder seus dados:
1. Feche o terminal que está executando o sistema.
2. Extraia todo o conteúdo deste arquivo dentro da pasta do Meu Fluxo.
3. Confirme a substituição dos arquivos existentes.
4. Execute: npm start
5. Atualize o navegador com Ctrl + F5.

IMPORTANTE: não altere a pasta src/. Ela contém os módulos da aplicação.
Se você quiser editar o sistema, comece por README.md e docs/ARQUITETURA.md.
